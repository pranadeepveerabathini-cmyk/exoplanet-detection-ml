# Exoplanet Detection using Machine Learning

Detecting exoplanets from NASA Kepler mission data using Random Forest classifier.

## Results
- Accuracy: 90%
- Dataset: NASA Kepler (9,564 candidates)
- Confirmed exoplanets: 2,293
- Key feature: Planet radius (koi_prad)

## Tools Used
Python, Scikit-learn, Pandas, Matplotlib, Seaborn
